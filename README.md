# ImageBord Webapp

本アプリは、[Recursion](https://recursionist.io/)のWeb Server With Databasesコースにおける制作物となります。

## 概要
ユーザー登録不要かつ自由にスレッドを立てて、さまざまな人とに意見を交流し合える場を提供するサービスです。
画像の保存は、ファイル名をpost_idとcreated_atをハッシュ化し、そのハッシュの先頭2文字を親ディレクトリにすることで
スケーラビリティを意識した構成となっています。

## 特徴
- **シームレスな利用体験**: ユーザー登録の手間が不要で、わかりやすいUI画面を提供しています。直感的な操作で、誰でも簡単に利用できます。

- **スケーラビリティ**: 画像はスケーラビリティを意識した構築をしており、さまざまなデバイスや解像度での閲覧に対応しています。

- **多様な拡張子に対応**: png、jpeg、jpg の拡張子に対応しており、さまざまな形式の画像をアップロードできます。

## どんなことができるの？
- ユーザー登録不要で自由に自分が興味あること・好きなことを話し合えるスレッドを立てることができます！
- お気に入りの画像をサムネイルにすることもできる！
- シンプルな画面UIなので、迷うことなく始められる！
